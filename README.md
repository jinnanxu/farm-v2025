# 智慧农场小程序2025

#### 介绍
小红书分享的原创毕业设计，SpringBoot+Vue.js+微信小程序

#### 软件架构
前端：微信原生小程序、Vue.js 
后端：SpringBoot 
数据库：MySQL 


#### 安装教程

1.  创建数据库farm，招行数据库脚本farm.sql 
2.  将farm目录导入微信开发者工具，修改你自己的appid，运行小程序端
3.  将farm-api目录导入idea，修改数据库连接串，启动后端接口
4.  将farm-vue导入VSCode，npm install安装依赖，npm run dev启动管理后台的前端Vue工程
